# Shoot Em Dead



# Instructions:
* Controls: 
    * Arrow keys or WASD to move the player ship up down left and right
    * Space bar or left mouse click to shoot. Hold down to shoot continuously ( Don't want you to get capel tunnel now)

Core mechanic:
Shoot til you die.
